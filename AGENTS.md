# SnakeSaintGer Agent Notes

This repository contains a minimal browser-based Snake game built with plain HTML, CSS, and JavaScript. The game has no build step and no external dependencies.

For future Codex tasks:
- Keep changes small, focused, and easy to review.
- Keep the project dependency-free unless the user explicitly asks otherwise.
- Prefer simple browser-native HTML, CSS, and JavaScript.
- Preserve the clean, minimal design and the Saint-Germain-en-Laye map-inspired game board.
